## Datasets Used

This project uses publicly available medical imaging datasets:

### 1. Breast Cancer (Histopathology)
- Dataset: **BreaKHis**
- Classes: Benign, Malignant
- Source: https://www.kaggle.com/datasets/ambarish/breakhis

Expected folder structure:
ml/data/breakhis/
├── benign/
└── malignant/

---

### 2. Cervical Cancer (Pap Smear Images)
- Dataset: **SIPaKMeD**
- Classes:
  - Normal: Metaplastic, Parabasal, Superficial-Intermediate
  - Abnormal: Dyskeratotic, Koilocytotic
- Source: https://www.cs.uoi.gr/~marina/sipakmed.html

Expected folder structure:
ml/data/SIPaKMeD/
├── normal/
└── abnormal/
